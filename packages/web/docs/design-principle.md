# Design Principle

**Note**: For complete design system implementation details, refer to `design-system.md`

## 기술 스택

### UI 프레임워크
- **TailwindCSS**: 유틸리티 우선 CSS 프레임워크로 일관된 디자인 시스템 구축
- **DaisyUI 5**: TailwindCSS 기반 컴포넌트 라이브러리로 빠른 UI 개발

## 핵심 디자인 철학

### 색상 사용 최소화 원칙
- **중성적 색상 팔레트 우선**: 흰색, 회색 계열을 기본으로 사용
- **포인트 색상 제한적 사용**: 브랜드 그라데이션을 메인 액센트로만 활용
- **기능적 색상 사용**: 색상은 정보 전달과 사용자 행동 유도 목적으로만 사용
- **브랜드 일관성**: 그라데이션을 브랜드 아이덴티티의 핵심 요소로 활용

### 그라데이션 활용 가이드
- **Primary CTA 버튼**: "시작하기" 등 주요 액션 버튼
- **텍스트 하이라이트**: 핵심 키워드 강조
- **진행률 표시**: 진행도, 완료 상태 등 동적 요소
- **호버 효과**: 인터랙티브 요소의 활성화 상태

## 레이아웃 원칙

### 2-Column 레이아웃 구조
- **Left Section**: 텍스트 콘텐츠와 CTA 버튼 집중 배치
- **Right Section**: 시각적 미리보기 (앱 인터페이스, 이미지 등)
- **균형감 있는 구성**: 텍스트와 시각적 요소의 적절한 비율 유지

### 헤더 디자인
- **로고 위치**: 좌측 상단에 브랜드 아이덴티티 강조
- **네비게이션**: 중앙 또는 우측에 기능별 메뉴 배치
- **액션 버튼**: 우측에 주요 액션 배치

## 타이포그래피 시스템

### 텍스트 계층 구조
- **메인 헤드라인**: 대형 볼드체로 강한 임팩트 전달
- **서브헤드라인**: 중간 크기로 부가 설명 제공
- **본문 텍스트**: 가독성 높은 크기와 줄간격
- **보조 텍스트**: 작은 크기로 추가 정보 제공

### 텍스트 강조 방법
- **색상 강조**: 그라데이션으로 핵심 키워드 하이라이트
- **굵기 조절**: 볼드체로 중요도 구분
- **크기 차별화**: 계층적 정보 구조 표현

## 버튼 디자인 시스템

### Primary Button (메인 액션)
- **배경**: 그라데이션
- **텍스트**: 흰색, 명확한 액션 지향적 문구
- **사용처**: "시작하기" 등 주요 액션

### Secondary Button (보조 액션)
- **배경**: 흰색 또는 투명
- **텍스트**: 진한 회색 또는 검은색
- **테두리**: 얇은 회색 테두리 (선택적)
- **사용처**: "더 알아보기" 등

### Ghost Button (최소 액션)
- **배경**: 투명
- **텍스트**: 중간 톤 회색
- **사용처**: "취소" 등

## 인터랙션 디자인

### 호버 효과 원칙
- **액션 가능 요소만 강조**: 버튼, 링크, 클릭 가능한 카드에만 호버 효과 적용
- **비액션 요소 호버 최소화**: 정보 표시용 카드는 호버 효과 제거 또는 최소화
- **과도한 꾸밈 자제**: 드롭 섀도우, 스케일, 트랜스폼 등 과도한 효과 피하기

### 포커스 상태
- **키보드 네비게이션**: 명확한 포커스 링 표시
- **접근성**: 충분한 대비와 시각적 피드백

## 반응형 디자인

### 모바일 최적화
- **단일 컬럼 레이아웃**: 모바일에서는 세로 배치
- **터치 친화적**: 버튼 크기와 간격 최적화
- **가독성**: 작은 화면에서도 명확한 텍스트 표시

### 태블릿/데스크톱
- **2-컬럼 레이아웃**: 넓은 화면 활용
- **호버 효과**: 마우스 인터랙션 최적화
- **여백 활용**: 넉넉한 공간으로 여유로운 느낌

## 접근성 고려사항

### 색상 대비
- **WCAG AA 준수**: 충분한 색상 대비 확보
- **색맹 고려**: 색상에만 의존하지 않는 정보 전달
- **다크모드 지원**: 어두운 배경에서도 가독성 확보

### 사용성
- **명확한 액션**: 버튼 텍스트가 의도를 명확히 전달
- **일관성**: 동일한 기능은 동일한 스타일 적용
- **피드백**: 사용자 액션에 대한 즉각적인 시각적 피드백
